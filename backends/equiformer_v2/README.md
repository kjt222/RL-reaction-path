# EquiformerV2 / FairChem backend (local checkout, not tracked)

This repo expects a local checkout of FairChem (ocp-models) under
`backends/equiformer_v2/fairchem/`. The source code is intentionally not
tracked in git.

- Source: https://github.com/FAIR-Chem/fairchem
- Version: 0.0.3 (from `setup.py` in the local checkout)
- Install: follow FairChem install docs; see `equiformerv2.yml` / `fairchemv2.yml`
