# MACE backend (local checkout, not tracked)

This repo expects a local checkout of MACE under `backends/mace/mace-torch/`.
The source code is intentionally not tracked in git.

- Source: https://github.com/ACEsuit/mace
- Version: 0.3.14 (from `mace/__version__.py` in the local checkout)
- Install: `pip install -e backends/mace/mace-torch` (see `mace_env.yml`)
